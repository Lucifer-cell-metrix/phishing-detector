# AI Phishing Detector — React Frontend

Modern React 19 application for the AI Phishing Detector using Vite.

## Features

- ⚡ Vite for fast development
- ⚛️ React 19 with JSX
- 🎨 Modern UI components
- 🔌 API integration with FastAPI backend

## Installation

```bash
cd frontend/react-app
npm install
```

## Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
```

## Linting

```bash
npm run lint
```
